CSS Inspection:

Check browser developer tools (F12):

  1. Inspect the image element
  2. Look at computed styles
  3. Check if image has width/height
  4. Check if .product-thumb has height

What you should see:

  • Yellow background = .product-thumb container
  • Red background = <a> tag
  • Green border = actual image

If you see yellow/red but NO green border, the image isn't loading.

If you see green border, the CSS positioning is the issue.
